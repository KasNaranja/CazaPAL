# Assets — CazaPAL / PAL España

Logos del proyecto. Fondo transparente. La palabra en blanco se ve sobre fondo oscuro.

Paleta:
- Rojo  `#e63946`
- Amarillo (gualda) `#f5c518`
- Blanco `#f5f5f5`
Tipografía del logotipo: **Anton** (Google Fonts).

## Archivos
- `mirilla-mark.svg` — solo la marca (mirilla). Vectorial, escalable, ideal para favicon/icono.
- `palespana-logo.png` — mirilla + `PAL`(blanco)`ES`(rojo)`PA`(amarillo)`ÑA`(rojo).
- `cazapalespana-logo.png` — mirilla + `CAZAPAL`(blanco)`ES PA ÑA`(rojo/amarillo/rojo).
- `caza-logo.png` — mirilla + `CAZA` (blanco).
- `precios-logo.png` — mirilla + `PRECIOS` (blanco).

Los PNG están exportados a 3x para uso nítido en web y retina.
